To Use:

Drag & Drop Main Menu & Pause Menu gameobjects into your relevant scenes.

To load into different scenes using the buttons, ensure you enter the name of the scene you wish to load in the relevant slot on the 'Main Menu Canvas > Main Menu (Script)' or the 'Pause Menu Canvas > Pause Menu (Script)'

To use the Audio Manager drag and drop the prefab into your main menu, it will stay active through all your scenes when the game is run from the main menu.

An audioTest scene is provided to demonstrate its use and placeholder audio is included with the Audio Manger prefab
